# ⚔ D&D Tracker

Локальное веб-приложение для ГМа - отслеживание HP, инициативы и статов существ во время боя в D&D и Pathfinder

## Возможности
- Добавление существ
- Автоматическая сортировка по инициативе
- Полоски здоровья с анимацией
- Временные HP не суммируются, берется большее значение
- Изменение КБ для урощения подсчета попадания при взятии щита или наложения заклинаний
- Смена инициативы, что б не добавлять заного персонажей в одной сессии
- Смена максимума хитов, как эффект повышения уровня во время сессии или бафов/дебафов
- Доступ с планшета через Tailscale или локально

## Стек

- **Backend**: Python 3.11+, FastAPI, SQLAlchemy 2.0 async, aiosqlite, SQLite
- **Frontend**: React 18, Vite

## Быстрый старт

Выходим в корневую папку программы и запускаем команду
```
<Полный путь к корневой папке>\.venv\Scripts\uvicorn.exe backend.main:app --host 0.0.0.0 --port 8001
```

Откроется на `http://localhost:8001`

### Доступ с iPad

Убедись, что iPad и компьютер в одной Wi-Fi сети, затем открой в браузере:

```
http://<IP-адрес-компьютера>:8001
```

Найти IP: `ipconfig` (Windows) или `ifconfig` / `ip a` (macOS/Linux)

## API

| Метод  | Путь                          | Описание                      |
|--------|-------------------------------|-------------------------------|
| GET    | /api/tracker                  | Список всех существ           |
| POST   | /api/tracker                  | Создать существо              |
| PATCH  | /api/tracker/{id}             | Обновить статы (КБ, иниц. и др.)|
| DELETE | /api/tracker/{id}             | Удалить существо              |
| POST   | /api/tracker/{id}/damage      | Нанести урон                  |
| POST   | /api/tracker/{id}/heal        | Применить лечение             |
